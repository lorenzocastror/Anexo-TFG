# Pipeline Automatizado para el Análisis Avanzado de la Morfología y Dinámica Mitocondrial

## Introducción
El presente repositorio contiene el conjunto de herramientas computacionales, scripts de procesamiento de imágenes y algoritmos de análisis estadístico desarrollados para la caracterización fenotípica de la estructura y motilidad mitocondrial. El pipeline está diseñado para ejecutar de manera secuencial la segmentación morfológica, el seguimiento espaciotemporal (*tracking*) de las organelas, la depuración de datos cuantitativos y la generación de representaciones gráficas de alta resolución para su posterior interpretación científica.

---

## Estructura del Pipeline y Documentación del Código

### 1. Configuración del Entorno Virtual (Environment)
Para garantizar la reproducibilidad del pipeline y la correcta ejecución de todas las dependencias matemáticas, de visión computacional y aprendizaje profundo, se requiere la preparación de un entorno virtualizado. La instalación de los paquetes necesarios se realiza mediante el gestor de paquetes `pip` ejecutando el siguiente comando:

```bash
pip install nellie torch trackastra napari openpyxl numpy pandas scikit-image scipy tqdm matplotlib opencv-python
```bash
